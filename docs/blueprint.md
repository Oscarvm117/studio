# **App Name**: Biotrazo Marketplace

## Core Features:

- User Authentication: Secure login and registration for both farmers and buyers with distinct user roles.
- Farmer Dashboard: A dashboard for farmers displaying key metrics like carbon reduction, created lots, total income, and emission reduction.
- Lot Creation Form: A form allowing farmers to create lot listings, including product type, quantity, unit (kg/lb/dozen), harvest date, location, price per kg (in Colombian Pesos), and certification selections.
- Lot Management: Farmers can view their available and sold lots in separate dashboard sections.
- Buyer Marketplace: Buyers can browse available lots from different farmers in a marketplace setting.
- Lot Search and Filtering: Buyers can search for specific lots manually with a filter function that shows the available lots.

## Style Guidelines:

- Primary color: Deep forest green (#228B22) to reflect nature and sustainability.
- Background color: Light beige (#F5F5DC), providing a neutral and earthy backdrop.
- Accent color: Golden yellow (#FFD700) to highlight important elements and calls to action.
- Body font: 'PT Sans', a humanist sans-serif for both body and headline
- Use icons that represent agricultural and sustainability themes, maintaining a clean and modern style.
- Minimalist and elegant design with clear sections for easy navigation and content discovery.
- Subtle animations for lot creation confirmations and marketplace interactions.
